# LLM Sandbox

A hands-on learning project for understanding how Large Language Models (LLMs) work.

## About

This project follows the book **[Build a Large Language Model (From Scratch)](https://www.manning.com/books/build-a-large-language-model-from-scratch)** by Sebastian Raschka, with accompanying code available at [rasbt/LLMs-from-scratch](https://github.com/rasbt/LLMs-from-scratch).
