# LLM Sandbox

A hands-on learning project for understanding how Large Language Models (LLMs) work.

## About

This project follows the book **[Build a Large Language Model (From Scratch)](https://www.manning.com/books/build-a-large-language-model-from-scratch)** by **Sebastian Raschka**, published by Manning (ISBN 978-1633437166, September 2024).

Accompanying code for the book is available at [rasbt/LLMs-from-scratch](https://github.com/rasbt/LLMs-from-scratch).
